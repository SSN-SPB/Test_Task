# Summarize Text

Create a concise summary of the provided text while preserving the key points.

## When to Use

- User asks for a summary.
- User asks for a TL;DR.
- Long documents need condensation.

## Input

One or more paragraphs of text.

## Output

A short summary in bullet points or paragraph form, capturing the essential information.

## How to Apply

Read the input text, identify the main points, and produce a brief summary. Prefer bullet points for distinct facts or figures; use a short paragraph when the content flows as a narrative.

## Examples

Input:
```
A long article describing quarterly company performance...
```

Output:
```
- Revenue increased by 12%.
- Operating costs decreased by 5%.
- Company raised full-year guidance.
```
