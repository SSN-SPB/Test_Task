<h1>Key parameters explained</h1>
<br>
num_results: Number of URLs to fetch<br>
<br>
lang: Search language (e.g., "en", "fr")<br>
<br>
sleep_interval: Seconds to wait between requests
<br>
advanced: Returns extra metadata (title, description)<br>
<br>
safe: Enable SafeSearch ("on" or "off")<br>
<br>
<br>
<h1>Summary</h1>
googlesearch is simple and fast for small scripts and learning<br>
Returns URLs (and optionally metadata)<br>
Not official, use with caution<br>
<br>
<br>
<h1>When NOT to Use This Library</h1>
High reliability<br>
Commercial use<br>
Many queries per day<br>
<br>
<h2>Instead, use:</h2>
Google Custom Search JSON API<br>
SerpAPI<br>
Bing Web Search API<br>
