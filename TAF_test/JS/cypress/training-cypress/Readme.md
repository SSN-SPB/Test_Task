# Cypress Training Kernel

---

## Purpose

This project is a complete Cypress Test Automation Framework (TAF)
created for learning and experimenting with Cypress.

Unlike most tutorials, this repository is built incrementally,
starting from a simple Express application and gradually introducing
Cypress features used in real enterprise projects.

The primary focus is learning:

- cy.intercept()
- Page Object Model
- Fixtures
- Custom Commands
- API Testing
- Request Inspection
- Response Modification
- Network Errors
- Loading Delays
- GitHub Actions (later)

---
